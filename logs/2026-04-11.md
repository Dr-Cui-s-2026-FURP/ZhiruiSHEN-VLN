## 1. 当前目标与范围

当前实现目标：先跑通本地链路，不接外部 API。

执行范围：
- 输入：ROS 2 话题 `/vln_instruction`（`std_msgs/String`）
- 推理：本地 Qwen3-VL-2B-Instruct（4-bit 量化）
- 解析：模型输出先尝试 JSON 坐标，失败则回退关键词规则
- 输出：直接调用 Nav2 `goToPose` 执行导航

暂不做：
- 外部 API 调用（仅保留占位文件）
- 英文说明文档

## 4. 如何启动（重启后照抄）

> 前提：Isaac Sim、Nav2、TF、点云转激光等基础链路已按 `reset.md` 启动。

### 4.1 构建包（首次或改代码后）

```bash
cd /home/bluepoisons/Desktop/FURP/VLN/ZhiruiSHEN-VLN
colcon build --packages-select vln_nav2_bridge
source install/setup.bash
```

### 4.2 启动 Node 5 桥接节点

```bash
conda activate isaaclab
source /opt/ros/humble/setup.bash
cd /home/bluepoisons/Desktop/FURP/VLN/ZhiruiSHEN-VLN
source install/setup.bash
ros2 run vln_nav2_bridge vln_node_local --ros-args -p use_sim_time:=True
```

### 4.3 发送一条测试指令

```bash
source /opt/ros/humble/setup.bash
cd /home/bluepoisons/Desktop/FURP/VLN/ZhiruiSHEN-VLN
source install/setup.bash
ros2 topic pub /vln_instruction std_msgs/msg/String "{data: 'Go to the right shelf with purple boxes'}" -1
```
