## 2026-03-13: 节点 2 文献综述与模型选择完成
* **调研范围**：精读 2023–2026 年间 VLN 领域 5 篇核心论文，聚焦连续环境（VLN-CE）下的感知、记忆与决策机制。
* **核心文献**：
  * *BEVBert* (2023) — 提出混合地图预训练范式，将局部度量地图与全局拓扑地图结合，增强空间推理能力。
  * *MapNav* (2025) — 引入标注语义地图（ASM），将视觉特征转化为文本标签，实现恒定内存占用（0.17 MB），解决长程导航遗忘问题。
  * *Uni-NaVid* (2025) — 基于 VLA 架构，通过 Token 合并实现 5 Hz 实时推理，支持 egocentric 视频流直接输入。
  * *Progress-Think* (2025) — 引入语义进度推理，动态对齐指令与视觉历史，有效解决停止决策故障和过冲问题。
  * *CMMR-VLN* (2026) — 利用多模态经验检索（RAG），使智能体能从历史失败轨迹中反思并优化当前策略。
* **模型选择**：确定构建**层次化语义增强 VLA 架构（HSA-VLM）**：
  * *感知层*：利用预训练多模态大模型（如 InternVL / LLaVA 变体）解析长指令。
  * *记忆层*：采用 ASM 策略，以低存储成本维持长期空间一致性。
  * *决策层*：采用 Uni-NaVid 流式 VLA 范式，确保 Habitat 连续环境下的平稳控制与低延迟响应。
  * *自反思层*：集成进度推理模块，显式监控指令完成度以提高任务成功率。
* **结论**：通过学术论证确定了模型技术路线，解决了连续环境下记忆衰减与指令对齐的矛盾。

## 2026-03-13: Node 2 Literature Review & Model Selection Completed
* **Survey Scope**: In-depth review of 5 core VLN papers (2023–2026), focusing on perception, memory, and decision-making in continuous environments (VLN-CE).
* **Key Papers**:
  * *BEVBert* (2023) — Proposed a Hybrid Map pre-training paradigm combining local metric maps with global topological maps, enhancing spatial reasoning.
  * *MapNav* (2025) — Introduced the Annotated Semantic Map (ASM), converting visual features into text labels to achieve constant memory usage (0.17 MB) and mitigate long-horizon forgetting.
  * *Uni-NaVid* (2025) — A VLA (Vision-Language-Action) architecture achieving 5 Hz real-time inference via token merging, supporting direct egocentric video stream input.
  * *Progress-Think* (2025) — Introduced semantic progress reasoning to dynamically align instructions with visual history, addressing the common VLN failure modes of stop-decision errors and overshooting.
  * *CMMR-VLN* (2026) — Leveraged multimodal experience retrieval (RAG) to enable the agent to reflect on past failed trajectories and refine its current strategy.
* **Model Selection**: Decided to build a **Hierarchical Semantics-Augmented VLA architecture (HSA-VLM)**:
  * *Perception layer*: Pretrained multimodal LLM (e.g., InternVL / LLaVA variants) for long-instruction parsing.
  * *Memory layer*: ASM strategy to maintain long-term spatial consistency at minimal storage cost.
  * *Action layer*: Uni-NaVid streaming VLA paradigm for smooth, low-latency control in Habitat continuous environments.
  * *Self-reflection layer*: Progress monitoring module to explicitly track instruction completion rate and improve task success.
* **Conclusion**: Resolved the contradiction between memory decay and instruction alignment in continuous environments through academic validation of the proposed architecture.
